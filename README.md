# 코틀린 기반 안드로이드 모빌리티 App 개발을 위한 스마트 모빌리티 전문가 양성과정 실습 레포
## Desktop Environment
- Arch Linux 6.6.10
- Android Studio Hedgehog

## Build Dependencies
1. google maps api 에서 maps_sdk_android 활성화, places_api 활성화
2. gradel의 local_properties파일에 api_key = "사용자 구글 맵 api 키" 추가
